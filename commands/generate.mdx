## Overview

The `generate` command creates cryptographically secure passwords and passphrases. It includes strength analysis, entropy calculation, and multiple generation modes.

<Note>
All passwords are generated using cryptographically secure random number generation (CSPRNG).
</Note>

## Usage

```bash
BLANK generate [options]
```

**Alias:** `gen`

## Options

<ParamField path="-l, --length" type="number">
  Password length (4-256 characters). Default: 20
</ParamField>

<ParamField path="-p, --preset" type="string">
  Use a preset configuration: `strong`, `standard`, `alphanumeric`, `memorable`, or `pin`
</ParamField>

<ParamField path="--passphrase" type="boolean">
  Generate a memorable passphrase instead of a password
</ParamField>

<ParamField path="-w, --words" type="number">
  Number of words for passphrase (3-12). Default: 5
</ParamField>

<ParamField path="-c, --copy" type="boolean">
  Copy generated password to clipboard
</ParamField>

<ParamField path="--no-symbols" type="boolean">
  Exclude symbols from password
</ParamField>

## Interactive Mode

If you run `generate` without options, it enters interactive mode:

```bash
BLANK generate
```

```
  🔐 Password Generator

  Generation mode:
  ❯ 🔐 Password (random characters)
    📝 Passphrase (memorable words)
    ⚡ Quick (use preset)
```

<Tip>
Interactive mode is the easiest way to explore all generation options.
</Tip>

## Presets

### Available Presets

| Preset | Length | Character Types | Use Case |
|--------|--------|----------------|----------|
| `strong` | 24 chars | Uppercase, lowercase, numbers, symbols | Maximum security |
| `standard` | 16 chars | Uppercase, lowercase, numbers, symbols | General use |
| `alphanumeric` | 16 chars | Uppercase, lowercase, numbers | No symbols allowed |
| `memorable` | 12 chars | Easy to type characters | Mobile-friendly |
| `pin` | 6 chars | Numbers only | PIN codes |

## Strength Analysis

### Strength Levels

| Score | Level | Bar Color | Meaning |
|-------|-------|-----------|----------|
| 0-20  | Weak | 🔴 Red | Very unsafe |
| 21-40 | Fair | 🟠 Orange | Marginally safe |
| 41-60 | Good | 🟡 Yellow | Reasonably safe |
| 61-80 | Strong | 🟢 Green | Safe |
| 81-100 | Excellent | 💚 Bright Green | Very safe |

### Entropy Calculation

Entropy measures password randomness in bits:

- **< 28 bits**: Very weak (easily cracked)
- **28-35 bits**: Weak
- **36-59 bits**: Fair
- **60-127 bits**: Strong
- **128+ bits**: Excellent

### Feedback and Suggestions

For passwords below "Strong" level, the generator provides actionable feedback:

```
  Strength:  ████░░░░░░░░░░░░░░░░ FAIR (38/100)
  Entropy:   42.3 bits
  Length:    8 characters

  Suggestions:
    • Increase length to at least 12 characters
    • Add symbols for better security
```

## Advanced Options

### Character Type Selection (Interactive)

In interactive mode, you can select exactly which character types to include:

```
  Character types:
  ❯ ◉ Uppercase (A-Z)
    ◉ Lowercase (a-z)
    ◉ Numbers (0-9)
    ◯ Symbols (!@#$...)
```

Deselect symbols by pressing Space.

### Ambiguous Character Exclusion

Exclude characters that look similar:

- `0` (zero) vs `O` (letter O)
- `1` (one) vs `l` (lowercase L) vs `I` (uppercase i)

This is useful for passwords you need to type manually.

## Use Cases

### Web Accounts (Strong)

```bash
BLANK generate --preset strong --copy
```

Generate a 24-character password for important accounts.

### Mobile Apps (Alphanumeric)

```bash
BLANK generate --preset alphanumeric --copy
```

No symbols for easier mobile keyboard typing.

### Master Passwords (Passphrase)

```bash
BLANK generate --passphrase --words 7
```

Memorable but strong passphrase for master passwords.

### Wi-Fi Password (Memorable)

```bash
BLANK generate --preset memorable
```

Easier to share verbally or type on devices.

### PIN Codes

```bash
BLANK generate --preset pin
```

Numeric PIN for devices or systems that require numbers only.

## Integration with `add` Command

When adding a new password entry, you can generate a password inline:

```bash
BLANK add
```

```
  Password:
  ❯ 🔐 Generate strong password
    📝 Generate passphrase
    ✏️  Enter manually

  [Select "Generate strong password"]

  Generated: xK9$mN2@pQ7#vB4!wL6%

  Use this password? (Y/n)
```

When using `--copy`, the password is automatically copied to your clipboard:

```bash

```

```

```

<Warning>
The clipboard contains sensitive data. Be careful when pasting.
</Warning>

## Technical Details

- Uses `crypto.randomBytes()` for cryptographically secure randomness
- Password character set: 94 printable ASCII characters
- Passphrase wordlist: 7,776 words (EFF long wordlist)
- Entropy formula: `log2(charset_size ^ length)`
- Strength scoring: Based on zxcvbn algorithm
- Excludes ambiguous characters when requested: `0O1lI`
